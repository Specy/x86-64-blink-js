# x86-64-blink-js
An emulator for x86-64 based off https://x64.halb.it


To build from source you must have installed:

- node
- texinfo
- emsdk
- musl-gcc
- make
- gcc
- flex
- bison


## Notes

To test this:

1. `git clone https://github.com/robalb/x86-64-blink-js/`
1. `npm run build:init-blink`
1. `npm run build:blink`
1. `npm run test`
